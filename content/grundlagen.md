# Einführung ins JavaScript

## JavaScript – wichtige Grundlagen

Heutzutage sind die meisten Websites **interaktiv**.  
Das bedeutet, dass Besucher mit der Seite interagieren können, zum Beispiel durch:

- **Slider** – interaktive Bilderkarusselle
- **Bildergalerien** – Navigation durch mehrere Bilder
- **Animationen** – bewegte Effekte auf der Seite
- **Buttons**, die Aktionen auslösen

Diese Interaktionen werden hauptsächlich durch **JavaScript** ermöglicht.

### Wichtige Aufgaben von JavaScript

- **Verarbeitung von Benutzereingaben** – z. B. Formulare, Klicks, Tastatureingaben
- **Reaktion auf Ereignisse (Event Handling)** – z. B. Klick, Mausbewegung, Tastendruck
- **Steuerung interaktiver Elemente** – dynamische Menüs, Slider, Modals
- **Kommunikation mit Servern** – Daten senden und empfangen ohne die Seite neu zu laden (AJAX / Fetch)

Damit reicht JavaScript von **einfachen Klick-Aktionen** bis hin zu **komplexen Funktionen in modernen Web-Apps**.

➡️ Deshalb gehört JavaScript heute zu den **wichtigsten Technologien im Web** und prägt stark, wie wir Websites und Online-Anwendungen erleben.

## Java vs. JavaScript – kurz

- **Unterschiedliche Sprachen:** Java und JavaScript wurden unabhängig entwickelt. Syntax und Einsatzgebiete unterscheiden sich.  
- **Namensähnlichkeit:** Marketing-Grund – JavaScript (1995) sollte vom damals populären Java profitieren.  
- **Entwicklung:** Beide Sprachen haben eigene Communities und Ökosysteme.  
- **JavaScript Standard:** Folgt **ECMAScript (ES)**; neueste Version: **ES 2023**.  
- JavaScript (JS) ist eine weit verbreitete Programmiersprache, erstellt von **Brendan Eich in 1995**.

## First JavaScript Program – kurz

Programme in JavaScript nennt man **Scripts**.  
Hier ein einfaches Beispiel:

```js
let name = "Katarzyna";
console.log(`Hallo ${name}, willkommen in der Welt von JavaScript!`);
```
